Power BI dashboard analyzing hospital ER patient flow, satisfaction, and waiting time.

# Hospital ER Analytics Dashboard (Power BI)

## Project Overview

This project presents a Power BI dashboard designed to analyze hospital emergency room operations.
The dashboard provides insights into patient distribution, satisfaction levels, waiting times, and demographic patterns.

The goal is to help hospital administrators understand operational performance and identify opportunities to improve patient experience.

---

## Key Business Questions

1. Which departments receive the highest number of patients?
2. What is the gender distribution of patients?
3. Which departments provide the highest patient satisfaction?
4. Which departments have the longest waiting times?

---

## Dashboard Metrics

### Total Patients

Total number of patient visits recorded in the dataset.

### Average Satisfaction Score

Average patient satisfaction rating across departments.

### Average Wait Time

Average waiting time before treatment.

---

## Visualizations

### Patient Distribution by Department

Shows which hospital departments receive the most patients.

### Gender Distribution

Displays the percentage distribution of male and female patients.

### Average Patient Satisfaction by Department

Highlights which departments provide the best patient experience.

### Average Wait Time by Department

Identifies departments with longer patient waiting times.

---

## Key Insights

* General Practice handles the largest number of patients.
* Patient gender distribution is nearly balanced between male and female.
* Gastroenterology shows the highest satisfaction scores.
* Neurology has the longest average waiting time.

---

## Tools Used

* Power BI
* Data Modeling
* Data Visualization
* Healthcare Operations Analysis

---

## Dataset

The dataset contains information on:

* patient_id
* patient_gender
* patient_age
* department_referral
* patient_waittime
* patient_sat_score
* doctor information
* billing data
